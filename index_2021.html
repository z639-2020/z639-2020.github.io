The course syllabus includes a course description and a list of all course policies.

Session 1—Week of 23. August

Before Class

  • Read: Pew Research Center. 2019. “Demographics of Social Media Users and Adoption in the United States.” Pew Research Center: Internet, Science & Tech (blog). June 12, 2019. https://www.pewinternet.org/fact-sheet/social-media/.
  • Review: Software Carpentry. "The Unix Shell".
  • Review: Software Carpentry. "Version Control with Git"
  • Those starting Python programming, start reading: Automate the Boring Stuff with Python. The book is available online and in print.
  • Those with more programming experience, start reading: A Whrilwind Tour of Python. The book is available online and in print.
  • Those wanting a rapid Python refresher, read: 1001 Lines of Python (Jupyter Notebook, Canvas Files Section).

During Class

  • Introduction to the course
  • Introduction to the computing environment
  • Introduction to Python
  • Assignments:
    • Sign up for presentations by the end of the first day of class. Use this form.
    • Create a GitHub account if you do not have one already. Feel free to create an account just for this course. Programming assignments are submitted using GitHub.
    • Assignment 1 distributed. Due Sunday! This assignment is intended to help you get comfortable with the autograding system. Please submit whatever work you have finished before the end of the week.

Session 2—Week of 30. August

Before Class

  • Read and be prepared to discuss:
    • Clark et al. 2018. “Using Twitter to Study Public Discourse in the Wake of Judicial Decisions: Public Reactions to the Supreme Court’s Same-Sex-Marriage Cases.” Journal of Law and Courts 6 (1): 93–126.
    • Sloan, Luke, and Jeffrey Morgan. 2015. “Who Tweets with Their Location? Understanding the Relationship between Demographic Characteristics and the Use of Geoservices and Geotagging on Twitter.” PLOS ONE 10 (11): e0142209. https://doi.org/10.1371/journal.pone.0142209.
  • Read or review:
    • Automate the Boring Stuff with Python. Chapters 1–7. (For those not particularly familiar with Python.)
    • Humanities Data Analysis. Chapters 1,3. (For those with fairly familiar with Python.)
    • Whirlwind Tour of Python. Notebooks 1–8. (For those very familiar with Python.)

During Class

  • Discuss the reading.
  • Lecture: Python basics (continued), Strings, Regular Expressions, Tokenization
  • Assignment 2 distributed. Assignment 2 due end of Week 3.

Session 3—Week of 6. September

Before Class

  • Read and be prepared to discuss:
    • Stewart, L. G., Arif, A., & Starbird, K. (2018). Examining trolls and polarization with a retweet network. In Proc. ACM WSDM, workshop on misinformation and misbehavior mining on the web.
    • Starbird, K., Arif, A., & Wilson, T. (2019). Disinformation as collaborative work: Surfacing the participatory nature of strategic information operations. Proceedings of the ACM on Human-Computer Interaction, 3(CSCW), 1-26.
  • Read or review:
    • Automate the Boring Stuff with Python. Chapters 8–9, 16. (For those not particularly familiar with Python.)
    • Notebooks 9-17 from VanderPlas, Jake (2016). A Whirlwind Tour of Python. O'Reilly. 978-1-491-96465-1. (For those with greater familiarity with Python.)

During Class

  • Discuss the reading.
  • Assignment 1 Discussion
  • Lecture: Text Representations
