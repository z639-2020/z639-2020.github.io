Using Python on Luddy Linux Servers

Note: the Linux computing environment changes somewhat from year to year. Please send corrections and updates to this document to riddella at indiana.

Note: This is suddenly out of date. Luddy installed brand new machines a few weeks ago. This page will be updated shortly.

Python 3 is installed on all of the Linux computers associated with Luddy. These computers are available for you to use and they are very powerful, modern workstations. If you are performing computationally-intensive work, you will find these computers will perform very well.

Accessing the Cluster

"At IU, what is the Burrow?" describes the linux cluster available to students in the School of Informatics and Computing.

You can access a computer on the cluster from anywhere in the world by using an SSH client such as ssh on macOS and Linux. Just type:

ssh <net-id>@burrow.sice.indiana.edu

where <net-id> should be replaced by your IU netid. If your netid is chen then you would run ssh chen@burrow.soic.indiana.edu.

If you find you do not have an account on the computers in the "burrow" cluster, e-mail your instructor.

(You may also access Linux computers (i.e., desktop workstations) by going to Luddy Hall 3111.)

Using Python

Running python3 will start the familiar Python REPL. You can run python scripts with python3 scriptname.py where scriptname.py is the name of a file containing Python code.

Using IPython

IPython is an enhanced REPL. You can install it on the computers in the cluster by running:

python3 -m pip install --user IPython

and launch it with:

python3 -m IPython

Installing Scientific Computing Packages

Many of commonly used packages are already installed on the cluster computers. To install all of the commonly used packages, run the following command:

python3 -m pip install --user numpy scipy nltk scikit-learn pandas

which will install the following packages: numpy, scipy, nltk, scikit-learn, and pandas.

Using a newer version of python

The IU servers tend to use older versions of python. If you would like to use, say, Python 3.7 you can enable it with the following:

module load python/3.7

You can add this command to your .bashrc to run this command automatically every time you log in. (Commands in your .bashrc are executed when you log in.)

Using Jupyter Notebook

If you are new to Python and wish to use Jupyter Notebook you likely want to install Python on your local machine. The following instructions are for experienced Linux users.

Using Jupyter Notebook in Lindley Hall

If you are in Luddy Hall 3111 using the Linux computers there, you can use a browser-based REPL, Jupyter Notebook.

First, install (or upgrade) the jupyter notebook with the following command:

python3 -m pip install --user jupyter

After Jupyter is installed, you can launch it with the following command:

python3 -m notebook

At this point, you should be able to use the jupyter notebook by opening a web browser and navigating to http://localhost:8888.

Using Jupiter Notebook Remotely

You can use the Jupyter Notebook remotely from any macOS, Linux, or Windows computer. To do this you need to create an ssh tunnel between your computer and the SOIC linux computer. This tunnel will allow you to access a port on the remote computer as if it were a local port.

The following instructions should work on both macOS and Linux machines. Windows users will need to consult documentation for their chosen SSH client.

First, pick a port number above 8000 which is unlikely to be chosen by another user. We will use 8891 in this example. If this port is in use on the remote (or local) computer, the following sequence of commands will not work. If there is a port conflict, pick another port and try again.

ssh -L 8891:localhost:8891 <netid>@burrow.soic.indiana.edu

Now, install (or upgrade) the jupyter notebook with the following command:

python3 -m pip install jupyter

Launch the jupyter notebook with the chosen port:

python3 -m notebook --port=8891

In the console you should see a line indicating how to access the notebook such as:

    Copy/paste this URL into your browser when you connect for the first time,
    to login with a token:
        http://localhost:8891/?token=ccc12f94863ee5ac8a22705c029fe5ae8b12512f9a4cde6

Paste this url into your local web browser.

When you are finished remember to shutdown the jupyter notebook with control-c and to close your ssh terminal.
